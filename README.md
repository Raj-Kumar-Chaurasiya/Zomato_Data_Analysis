# Zomato Data Analysis Project 🍽️📊

## Project Overview

This project focuses on analyzing Zomato restaurant data to understand customer preferences, restaurant categories, ratings, online ordering trends, and business patterns.

The analysis was performed using Python data analytics libraries to clean, process, visualize, and extract meaningful insights from the dataset.

---

## Objectives

- Analyze restaurant distribution by type
- Understand customer voting patterns
- Study restaurant ratings distribution
- Compare online and offline ordering trends
- Analyze approximate cost for two people
- Identify patterns between restaurant categories and customer behavior

---

## Dataset

The dataset contains information about Zomato restaurants including:

- Restaurant type
- Online ordering availability
- Customer ratings
- Number of votes
- Approximate cost for two people
- Restaurant categories

Dataset file:
`Zomato data .csv`

---

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning & Preparation

Steps performed:

- Loaded dataset using Pandas
- Checked dataset information and structure
- Converted rating values into numerical format
- Handled data formatting issues
- Prepared data for visualization and analysis

## Exploratory Data Analysis (EDA)

### 1. Restaurant Type Analysis

Created count plots to understand which restaurant categories are more common.

Insights:
- Identified the most popular restaurant types
- Compared restaurant availability across categories

---

### 2. Customer Votes Analysis

Analyzed total votes received by different restaurant types.

Visualization:
- Line plot

Purpose:
- Understand customer engagement across restaurant categories

---

### 3. Rating Distribution Analysis

Analyzed how restaurant ratings are distributed.

Visualization:
- Histogram

Insights:
- Identified common rating ranges
- Observed customer rating patterns

---

### 4. Cost Analysis

Analyzed approximate cost for two people.

Visualization:
- Count plot

Purpose:
- Understand pricing trends among restaurants

---

### 5. Online Order Analysis

Compared ratings based on online ordering availability.

Visualization:
- Box plot

Insights:
- Studied whether online ordering impacts customer ratings

---

### 6. Online Order Heatmap

Created a heatmap between restaurant types and online ordering.

Insights:
- Identified restaurant categories with higher online ordering adoption
- Compared ordering behavior across restaurant types

---

## Key Insights

- Restaurant categories have different levels of popularity among customers.
- Customer engagement varies significantly based on restaurant type.
- Online ordering availability shows different patterns across restaurant categories.
- Ratings help understand customer satisfaction trends.
- Cost analysis provides an overview of restaurant pricing behavior.
